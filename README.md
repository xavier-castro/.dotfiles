# README

## Quick Cheatsheet

Inside `Ex`: `o` opens the default application of highlighted file
`v` opens a vertical one (Same with my oil setup rn)
`x` opens horizontal

Harpoon `h` `t` `n` `s` are the keybinds. `<C-S-P>` and `<C-S-N>` for previous and next. `<leader>a` to Add
